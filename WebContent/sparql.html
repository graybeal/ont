SPARQL (Simple Protocol and RDF Query Language) is the standard query language for RDF (Resource Description Framework). More information here. You can use this form to submit your SPARQL query against the MMI Ontology Registry and Respository.
Both GET and POST methods are accepted.

CONSTRUCT examples SELECT examples

Note: The default output format for SELECT queries is HTML. To obtain an output in other format, add the "form" parameter in your GET or POST request, for example:
  • http://mmisw.org/ont/?form=json&sparql=SELECT ?s ?p ?o WHERE {?s ?p ?o. } LIMIT 20.
  • http://mmisw.org/ont/?form=csv&sparql=SELECT ?s ?p ?o WHERE {?s ?p ?o. } LIMIT 20.
  • http://mmisw.org/ont/?form=html&sparql=SELECT ?s ?p ?o WHERE {?s ?p ?o. } LIMIT 20.
